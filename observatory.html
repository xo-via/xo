The Universe — Observatory

A universe being built one primitive at a time. The folder is the universe. It began with a question: what if you could visualize your thoughts?
t=? ticks since the big bang · one tick = 10 minutes

The idea

A thought is meant to be stored as a folder, not a file — because a thought is recursive (it contains smaller thoughts), connected (it points at others), and alive in time (it is born on a tick). The first thing the universe needed before it could hold a thought was time, so time was built first.

What it has

Time, the first primitive

  • The big bangThe root commit is t=0. Time is derived from the distance to it.
  • Ticks every 10 minDuration lives in CONSTANTS as SNAPSHOT_DURATION_SECONDS=600 — the single source of truth.
  • Snapshots via gitEach tick commits an (allowed-empty) snapshot — the log is a literal timeline.

A self-sufficient engine

  • One entry pointpython3 xo.py observes the fundamentals and bootstraps everything.
  • Cron heartbeatxo.py snapshot runs every 10 min on its own; also callable by hand.
  • Mirrored off-siteEvery snapshot is pushed to the fork xo-via/xo. Authored by Satori.

What it needs

To fulfil the idea

  • Actual thoughtsThe thought-as-folder format is described but none exist yet — nothing has been thought.
  • The Visualizer, wired upA view has begun — the observe/ submodule (a Next.js app) — but it isn't yet reading the universe's real data.
  • Links between thoughtsThe synapses (thought → thought) that turn folders into a mind.

To be robust

  • More primitivesTime is the only fundamental so far. Space? Matter? Memory? The registry is ready for them.
  • Always-on timeCron only ticks while this container is awake; restarts need xo.py re-run.
  • Self-observationThe observatory should read live state from the repo, not from hard-coded facts.
